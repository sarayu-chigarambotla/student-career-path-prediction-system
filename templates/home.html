Random Forest Algorithm

Random Forest is an ensemble machine learning technique that combines multiple decision trees to improve prediction accuracy and reduce bias. It evaluates academic, technical, and behavioral attributes to determine the most suitable career path for a student.

  • ✔ Reduces overfitting compared to single models
